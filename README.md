# Full Featured Notepad
**Noxopad is a small and light notepad for windows OS**

C# - .NET

# Options :
- Change Fore Color (text color)
- Change background Color 
- Open - Save - Print documents
- text formant tools :
- - Font
- - Colors
- Edit tools :
- - copy - cut - paste - select all ...
